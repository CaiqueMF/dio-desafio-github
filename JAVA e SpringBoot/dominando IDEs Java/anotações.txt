Java SE/Java EE/ Java ME

código.java -> JDK -> código.class -> JVM

JDK vs JRE
JDK (Java Development Kit)
	é o kit de Desenvolvimento Java responsável por compilar código fornte .java em bytecode .class
JVM (JAva Virtual Machine)
	é  a máquina virtual do java responsável por executar o bytecode .class
JRE (Java Runtime Environment)
	ambiente de execução do java que forcnece as bibliotecas padrões do java para o JDK
